Options

(Specify Sections)
+[no]stats
+[no]answer
+[no]...

eg. $ dig nycpython.com +nostats
    ...
(Specify a file with multiple FQDNs)
-f <file>

eg. $ dig -f names.txt
    ...
(Just grab the IP)
+short

eg. $ dig nycpython.com +short
    54.204.41.210

Options

(Specify type of DNS record)
-t MX or MX

eg. $ dig nycpython.com -t MX
    ...

    $ dig nycpython.com MX
    ...
